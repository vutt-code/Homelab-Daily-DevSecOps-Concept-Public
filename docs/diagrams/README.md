# Diagrams

The diagrams use Mermaid so they render directly in GitHub Markdown.

They follow the project `c4-diagram` skill conventions using C4-style Mermaid flowchart notation. The structured C4 source note is:

- [`../System - Daily DevSecOps Concept.md`](../System%20-%20Daily%20DevSecOps%20Concept.md)

This is a C4 Lite representation rather than strict enterprise architecture notation. The goal is readability for a public architecture story.
