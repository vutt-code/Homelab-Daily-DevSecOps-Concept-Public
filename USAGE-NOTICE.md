# Usage Notice

This repository is published as a **portfolio and architecture reference**.

It is intended to show the design approach behind a homelab Daily DevSecOps, knowledge-base RAG, and evaluator workflow concept.

## Not a turnkey open-source solution

This repository intentionally does not include:

- full n8n workflow exports,
- production credentials,
- private topology,
- raw configuration exports,
- live logs,
- private wiki contents,
- or a complete deployment guide.

## No broad open-source license granted

Unless the author later adds a specific open-source license, this repository should be treated as **source-available reference material only**.

You may read it for learning and discussion. Do not assume permission to copy, redistribute, repackage, deploy, or present it as a complete reusable solution.

If you want to reuse substantial parts of the documentation, snippets, or architecture pattern, contact the author for permission or add your own implementation from first principles.

## Security note

The repository uses sanitized examples. Do not open public issues or pull requests that include secrets, internal IPs, workflow IDs, raw config files, or private telemetry.
